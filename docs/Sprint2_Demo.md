# Sprint 2 Demo – Financial Ratio Engine

## Project

Nifty100 Financial Analytics

## Sprint

Sprint 2 – Financial Ratio Engine

## Sprint Objective

Develop a financial ratio engine to calculate, validate, and store financial KPIs for all companies in the SQLite database.

## Features Demonstrated

### Financial Ratio Engine

* Net Profit Margin
* Operating Profit Margin
* Return on Equity (ROE)
* Return on Capital Employed (ROCE)
* Return on Assets (ROA)
* Debt-to-Equity Ratio
* Interest Coverage Ratio
* Asset Turnover Ratio

### CAGR Engine

* Revenue CAGR
* PAT CAGR
* EPS CAGR

### Cash Flow KPIs

* Free Cash Flow
* CFO Quality
* CapEx Intensity
* FCF Conversion
* Capital Allocation Classification

### Database Integration

* Populated `financial_ratios` table
* 1160 company-year records updated
* 21 database columns available
* Composite Quality Score generated
* ROCE integrated into SQLite

### Validation

* 37 unit tests passed successfully
* Ratio validation completed
* Edge case report generated
* Financial sector leverage carve-out verified
